# Nexora

## Overview

Nexora is a File Integrity Monitoring (FIM) platform built to detect unauthorized file modifications using SHA-256 hashing.

## Tech Stack

- Frontend: React
- Backend: FastAPI
- Language: Python
- Database: PostgreSQL
- Monitoring: Watchdog
- Hashing: SHA-256
- Version Control: Git & GitHub
